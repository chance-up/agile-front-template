# agile-front-template

## Code Convention

### 1. VSCode Setting -> format on Save 기능 On

- 저장 시 자동으로 prettier 규칙에 맞게 파일 format을 변경해준다.

### 2. Prettier Rule

- printWidth : 100  
  // 줄 바꿈 폭의 너비

- tabWidth : 2  
  // tab = space x 2

- singleQuote : true  
  // 문자열 작성 시 '문자열'

- semi : true  
  // 세미콜론 사용
